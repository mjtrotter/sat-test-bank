You are extracting math competition problems from MATHCOUNTS National Competition PDFs into structured JSON.

SOURCE: ~/Downloads/MathCounts Tests/{YEAR}/national/ — each year has sprint, target, team, and answers PDFs.

APPROACH:
1. Convert each PDF page to a 200dpi PNG using PyMuPDF (fitz). Save to /tmp/mc_national_{YEAR}/
2. Read each PNG image visually to extract problem text and answers
3. Match answers to questions by round + problem number
4. Validate counts: Sprint=30, Target=8, Team=10 — EXACTLY. No exceptions.
5. Write JSON to ~/Desktop/Dev/Vanguard/data/extracted/{YEAR}_national.json

CONVERT SCRIPT (run per year):
```python
import fitz, os
year = 2008
base = os.path.expanduser(f"~/Downloads/MathCounts Tests/{year}/national/")
out = f"/tmp/mc_national_{year}"
os.makedirs(out, exist_ok=True)
for fname in os.listdir(base):
    if not fname.endswith('.pdf'): continue
    prefix = fname.split()[0]
    doc = fitz.open(os.path.join(base, fname))
    for pg in range(len(doc)):
        pix = doc[pg].get_pixmap(dpi=200)
        pix.save(f"{out}/{prefix}_page{pg}.png")
    doc.close()
print(f"Converted {year} to {out}")
```
